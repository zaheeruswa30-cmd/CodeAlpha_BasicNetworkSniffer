# Basic Network Sniffer

## Description
This project is a Basic Network Sniffer developed using Python and the Scapy library as part of the CodeAlpha Cyber Security Internship. It captures live network packets and displays important information such as Source IP Address, Destination IP Address, Protocol, and Payload.

## Features
- Captures live network packets
- Displays Source IP Address
- Displays Destination IP Address
- Detects TCP, UDP, and ICMP protocols
- Displays packet payload
- Simple and easy to use

## Technologies Used
- Python 3
- Scapy

## Requirements
- Python 3.x
- Scapy

## Installation

1. Clone the repository:
```bash
git clone https://github.com/your-username/CodeAlpha_BasicNetworkSniffer.git
```

2. Navigate to the project folder:
```bash
cd CodeAlpha_BasicNetworkSniffer
```

3. Install the required library:
```bash
pip install scapy
```

## Usage

Run the following command:

```bash
python sniffer.py
```

**Note:** Run the terminal as Administrator to capture network packets.
